# Практична 10 (частина 1). Проміси та async/await
## Варіант 1: Система замовлення їжі (Promise chains)

## Функціонал
### Система симулює 4 етапи обробки замовлення їжі:
- **Перевірка наявності**
- **Резервування товарів**
- **Обробка оплати**
- **Призначення доставки**

### Кожен етап має **20% шанс виникнення помилки**, що дозволяє протестувати централізовану обробку через `.catch()`.

## Технології
- Native JavaScript Promises
- Promise Chain (`.then()`, `.catch()`, `.finally()`)
- HTML5/CSS3 для інтерфейсу

## Як запустити
- 1. Склонуйте репозиторій.
- 2. Відкрийте `index.html` у вашому браузері.
- 3. Натисніть кнопку "Створити замовлення".
