# Практична робота 7.1: Операції з масивами
## Варіант 1: Система управління студентами

## Технічна складова (Використання методів масивів)
-1. **`.filter()`** — застосовується для динамічного відсікання студентів за обраним факультетом та мінімальним середнім балом.
-2. **`.sort()`** — виконує гнучке сортування масиву об'єктів як за числовими полями (`age`, `grade`), так і за текстовими (`name` через `localeCompare`), підтримуючи режими ASC та DESC.
-3. **`.find()`** — реалізує функціонал "розумного" пошуку, знаходячи першого студента в базі, чиє ім'я частково або повністю збігається із запитом.
-4. **`.reduce()`** — виконує три аналітичні задачі:
-- Обчислює загальний середній бал усіх відфільтрованих студентів.
-- Знаходить об'єкт студента з максимальним показником `grade`.
-- Групує студентів за факультетами, створюючи карту розподілу.
