# Практична робота 7.2: Алгоритми з масивами
## Варіант 1: Алгоритми сортування

| Алгоритм           |   Best Case  |  Average Case |    Worst Case | Особливості |
|      :---          |     :---:    |     :---:     |      :---:    | :--- |
| **Bubble Sort**    |     O(n)     |    O(n^2)     |      O(n^2)   | Багато зайвих обмінів |
| **Selection Sort** |    O(n^2)    |    O(n^2)     |      O(n^2)   | Стабільна кількість порівнянь |
| **Insertion Sort** |     O(n)     |    O(n^2)     |      O(n^2)   | Ідеальний для майже відсортованих масивів |
| **Merge Sort**     |  O(n \log n) |  O(n \log n)  |   O(n \log n) | Потребує додаткової пам'яті |
| **Quick Sort**     |  O(n \log n) |  O(n \log n)  |      O(n^2)   | Найшвидший на практиці, але вразливий до невдалих опорних елементів |
