# Практична робота №6 (Частина 1): Advanced Contact Form
## Варіант 1: Advanced Contact Form

### Опис проєкту
Це інтерактивна контактна форма з розширеним функціоналом, створена для демонстрації роботи з подіями в JavaScript та покращення користувацького досвіду (UX). Проєкт реалізує "живу" перевірку введених даних, візуалізацію прогресу заповнення та оптимізацію продуктивності.

### Основні функції
- **Live-валідація:** Кожне поле перевіряється миттєво під час введення.
- **Прогрес-бар:** Динамічне відображення відсотка заповненості форми (0-100%).
- **Візуальний Feedback:** - Кольорова індикація (зелений/червоний) та іконки успіху/помилки.
  - Плаваючі лейбли (Floating labels) для сучасного вигляду.
- **Character Counter:** Лічильник символів для текстового повідомлення (ліміт 500 символів).
- **Smart Submit:** Кнопка відправки автоматично блокується, якщо хоча б одне поле не відповідає правилам.
- **Loading State:** Ефект завантаження на кнопці після натискання.

### Технічні особливості
- **Оптимізація (Debounce):** Використано функцію затримки для валідації, щоб зменшити кількість обчислень при швидкому введенні тексту.
- **Делегування подій (Event Delegation):** Оптимізована обробка подій для елементів форми.
- **Доступність (A11y):**
  - Використання семантичних тегів HTML5.
  - ARIA-атрибути для прогрес-бару та повідомлень про помилки.
  - Повна підтримка навігації клавіатурою (Tab-index).
- **Адаптивність:** Mobile-first дизайн, коректне відображення на будь-яких екранах.

### Правила валідації
- **Ім'я:** Тільки літери, мінімум 2 символи.
- **Email:** Стандартний формат електронної пошти.
- **Телефон:** Формат України (+380XXXXXXXXX).
- **Тема:** Обов'язковий вибір із випадаючого списку.
- **Повідомлення:** Від 20 до 500 символів.

### Структура файлів
- `index.html` — семантична розмітка з ARIA-атрибутами.
- `style.css` — стилі, анімації станів та адаптивна верстка.
- `script.js` — логіка валідації, debounce та обробка стану форми.
