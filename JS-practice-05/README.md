# User Management System

## Опис
Система управління користувачами.

## User
Кожен користувач представлений об'єктом зі наступними властивостями:
- id (string): Унікальний ідентифікатор.
- name (string): Ім'я користувача.
- email (string): Валідна електронна пошта.
- role (string): Роль в системі (наприклад, 'admin' або 'user').
### Методи:
- getInfo(): Повертає рядок з основною інформацією.
- isAdmin(): Повертає true, якщо роль користувача — "admin".
- updateProfile(data): Повертає новий об'єкт із оновленими даними, зберігаючи старий ID.

## UserManager
Керує колекцією користувачів через замикання (closures), що робить внутрішній масив users недоступним для прямої модифікації ззовні.
- createUser(data) — створює та додає нового користувача.
- getUser(id) — пошук користувача за ID.
- updateUser(id, data) — оновлення даних існуючого користувача.
- deleteUser(id) — видалення користувача зі сховища.
- getAllUsers() — отримання списку всіх користувачів.
- getUsersByRole(role) — фільтрація списку за роллю.
