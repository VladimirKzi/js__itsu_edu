# Практична робота №8: Функції вищого порядку
## Варіант 1: Data Transformation Library

### Опис проєкту
Ця бібліотека містить набір інструментів для функціонального програмування в JavaScript, реалізованих у межах Практичної роботи №8.

## Основний функціонал
- **Custom Iterators**: Власні реалізації `map`, `filter`, `reduce` без використання вбудованих методів `Array.prototype`.
- **Composition**: Функції `compose` та `pipe` для створення конвеєрів обробки даних.
- **Currying & Partial**: Інструменти для часткового застосування аргументів.
- **Memoization**: Кешування результатів функцій для оптимізації продуктивності.
- **Chainable API**: Зручний інтерфейс для послідовних трансформацій даних.
