# Практична 10 (частина 2). Проміси та async/await
## Варіант 1: GitHub User Explorer

## Функціонал
- Пошук будь-якого публічного користувача GitHub.
- Відображення аватара, біо та статистики.
- Список ТОП-10 репозиторіїв за кількістю зірок.
- **Оптимізація**: Паралельне завантаження профілю та списку репозиторіїв через `Promise.all`.
- Обробка помилок (404, перевищення лімітів API).

## Технології
- **Async/Await**: Повністю асинхронний код без `.then()`.
- **Fetch API**: Взаємодія з GitHub REST API.
- **Loading State**: Візуальний індикатор завантаження.

## Як запустити
- 1. Склонуйте репозиторій.
- 2. Відкрийте `index.html` у вашому браузері.
